# CampusMate

[立即下载 CampusMate](https://campusmate.cloud)

> 听不懂、跟不上、答不上。  
> CampusMate 让你在课堂、面试、答辩这些高压场景里，少一点慌，多一点把握。

CampusMate 是一款面向留学生与跨语言高压沟通场景的 macOS 实时辅助软件。

它不是普通翻译工具，也不是普通笔记软件。

它更像一个实时辅助层:

- 一边听，一边转成文字
- 一边看，一边同步翻译
- 需要回答时，立刻给你思路

如果你经历过这些时刻，你会立刻明白它的价值：

- 课堂上老师说太快，单词认识，但整句听不懂
- 面试时问题听清了，但脑子空白，来不及组织回答
- 答辩时老师刚开口，压力瞬间拉满
- 看外语视频时总要来回暂停、倒退、猜意思

CampusMate 做的，就是把这些真实又高频的压力往下拉。

## 它能帮你做什么

### 听课模式

适合 `上课`、`讲座`、`线下分享`。

- 实时记录课堂原文
- 实时显示翻译内容
- 支持向上回看完整记录
- 可按需整理 AI 总结与 Todo

你不需要再一边硬听一边慌着记重点。

### 面试模式

适合 `线上面试`、`模拟面试`、`岗位问答`。

- 实时识别面试官提问
- 原文和翻译同步显示
- 结合简历背景给出参考回答
- 帮你更快进入“能开口回答”的状态

不是替你死板背答案，而是在最关键的几秒钟里，给你支点。

### 答辩模式

适合 `课程答辩`、`项目汇报`、`presentation`。

- 可提前上传项目资料
- 实时识别老师问题
- 自动展示问题原文与翻译
- AI 给出更贴近你项目上下文的参考回答
- 支持历史问答回看

越是高压场景，越需要一个不会慌的辅助层。

### 追剧模式

适合 `外语视频`、`追剧`、`日常语言输入`。

- 系统内录实时识别
- 双语字幕贴底显示
- 进入后默认直接运行
- 纯本地链路，不依赖 AI

轻一点，但非常高频，也非常实用。

## 为什么它会让人上头

- 真正围绕留学生场景做，不是泛泛而谈的“AI 工具”
- 重点解决的是“临场压力”，不是单纯做个翻译框
- 面试、答辩、听课、追剧四种模式直接覆盖高频真实需求
- 本地识别、本地翻译优先，使用体验更直接
- 服务端 AI 只在真正需要的时候介入，把辅助做在关键点上

很多工具在展示功能。

CampusMate 更像是在解决一个很具体的问题：

当你明明有能力，但因为语言、速度、压力，发挥不出来的时候，能不能有一个东西在旁边托你一下。

- 要做答辩、presentation、项目汇报的人
