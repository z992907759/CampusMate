# CampusMate

[点击下载 CampusMate](https://campusmate.cloud)

> 面向留学生与高压实时场景的 macOS 实时辅助软件

CampusMate 是一款专为真实高压沟通场景设计的 macOS 应用。

它的目标很直接：

- 先帮你听懂
- 再帮你看懂
- 最后帮你更快组织出能说出口的内容

无论是课堂听课、线上面试、课程答辩，还是外语视频理解，CampusMate 都希望把“听不清、跟不上、答不上”的压力降下来。

## 为什么是 CampusMate

- 实时转录：把系统音频或麦克风内容快速转成文字
- 实时翻译：原文和目标语言同屏显示，理解更直接
- AI 即时辅助：在答辩、面试等场景中，结合上下文快速生成参考回答
- 本地优先：音频采集、本地识别、本地翻译尽量留在本机完成
- 场景化设计：不是通用聊天工具，而是围绕真实使用场景做出来的产品

## 四种核心模式

### 1. 答辩模式

适合 `课程答辩`、`项目汇报`、`presentation`。

- 支持提前上传项目文件
- 实时识别老师提问
- 自动展示问题原文与翻译
- AI 给出参考回答
- 支持历史问答回看

如果你在答辩时最怕“问题听明白了，但脑子一片空白”，这个模式就是为这种场景准备的。

### 2. 面试模式

适合 `线上面试`、`模拟面试`、`岗位问答`。

- 支持上传简历
- 可结合你的背景生成更贴近你的回答思路
- 实时识别面试官问题
- 原文、翻译、AI 回答同屏显示
- 更适合英文面试、跨语种沟通场景

不是替你机械回答，而是在你最需要的时候，给你一个更快组织语言的支点。

### 3. 听课模式

适合 `听课`、`讲座`、`课堂记录`、`线下分享`。

- 麦克风实时采集内容
- 连续显示课堂原文与翻译
- 可向上回看完整记录
- 支持按需整理 AI 总结与 Todo

对于“老师说得太快”“单词都认识但连起来听不懂”的场景，帮助会非常明显。

### 4. 追剧模式

适合 `外语视频`、`追剧`、`日常语言输入`。

- 系统内录实时识别
- 双语字幕贴底显示
- 进入模式后默认直接运行
- 本地链路，不依赖 AI

这个模式更轻，也更适合日常高频使用。

## 适合谁

- 正在海外读书、经常上英文课的留学生
- 需要做答辩、presentation、口头汇报的人
- 需要参加线上英文面试的人
- 想提升外语理解效率的人
- 希望把“听不懂”和“来不及组织回答”的焦虑降下去的人

## 当前体验重点

- 面试模式、答辩模式：更强调实时理解与 AI 回答辅助
- 听课模式：更强调原文、翻译、总结与记录沉淀
- 追剧模式：更强调轻量、顺手和高频使用

CampusMate 不是传统笔记软件，也不是普通翻译工具。

它更像一个在关键时刻站在你旁边的实时辅助层。

## 当前支持

- 平台：`macOS 15（Sequoia）及以上`
- 语言：支持多语言实时理解与显示
- 下载方式：官网直接下载 macOS 安装包

Windows 版本仍在规划中。

## 一句话总结

如果你经历过这些时刻：

- 听课时明明很努力，但就是跟不上
- 面试时知道自己会，却来不及组织语言
- 答辩时老师一开口，大脑瞬间空白

那么 CampusMate 就是为你做的。

它想做的不是“炫技”，而是把语言压力、理解压力和临场压力，真正往下拉一点。
